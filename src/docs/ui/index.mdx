## What It Is

The UI section explains how ApiHug extends contract-first design into the frontend layer. It covers the proto-facing model, Vue-oriented application structure, and the toolchain that turns generated metadata into usable UI code.

## When To Use It

Read this section when you want to:

- generate or structure a frontend application from ApiHug contracts
- understand how the UI module fits the backend project layout
- connect router, service, i18n, and generated metadata in one workflow
