## 适用场景

当你准备从 IntelliJ 里直接创建一个新的 ApiHug 项目，而不是手工拼装目录和构建文件时，使用这个流程。

## 操作步骤

1. 打开 IntelliJ，选择 `File -> New -> Project`。
2. 在项目向导中选择 `ApiHug`。
3. 填写项目基础信息：
   - package
   - project name
   - description
4. 选择初始化 SDK 与运行时选项：
   - ApiHug version
   - database vendor
   - cache
   - service port
5. 完成 Spring Boot 初始化向导：
   - 选择项目类型
   - 勾选需要的 starter，例如 `Spring Web`
6. 点击 `Create`，然后打开生成后的项目。

## 你会看到什么

向导会生成一套可直接使用的 ApiHug 项目骨架，包括基础模块布局、构建文件以及启动所需的初始配置。
