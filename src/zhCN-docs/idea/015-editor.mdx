## 本页说明

这一页介绍 ApiHug IntelliJ 插件提供的编辑器上下文操作。

当你希望直接在 IDE 中完成 proto 文档创建、结构补全和常见编辑动作，而不是手写所有模板代码时，请使用这些能力。

另请参考：[快捷键速查](/zhCN-docs/idea/000-shortcut)

## 适用场景

当你需要：

- 快速创建新的 proto 文件
- 为消息、字段、服务、枚举、错误码、实体添加标准结构
- 在 IDE 内持续完成契约设计，不反复切换到外部模板或脚本

就应该使用编辑器助手。

## 新建 Proto 文件

1. 打开 `proto` 模块中的 `src/main/proto`
2. 右键目标目录
3. 选择 `Add Proto File`
4. 选择正确的 proto 文件类型

<img src={require('@/img/plugin/115_menu_add_file.gif').default.src} alt="ApiHug 编辑器助手新增文件" width={2334} height={1768} className="rounded-lg" />

## 编辑 Message 文件

适用于声明了如下类型的文件：

```proto
option (hope.domain.kind) = BEAN;
```

常见操作包括：

- `Add Message`
- `Add Field`
- `Add Field(Reference)`
