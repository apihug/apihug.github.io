## Structure

```
├─assets (1)
│  ├─styles
│  │  └─transition
│  └─svg-icon
├─components (2)
│  ├─dashboard
│  └─ui
│      ├─accordion
│      ├─alert
├─composables (3)
├─config  (4)
├─directive (5)
├─layouts (6)
│  └─admin
├─lib (7)
├─locales  (8)
│  └─langs
│      ├─en-US
│      └─zh-CN
├─pages (9)
│  ├─dashboard
│  └─example
├─router (10)
│  └─auto
├─service (11)
│  ├─api
│  ├─form
│  ├─model
│  └─table
├─stores (12)
├─types  (13)
└─utils (14)
```

| Directory | Content               |
|-----------|-----------------------|
| (1)       | CSS Styles            |
| (2)       | Component Library     |
| (3)       | Hooks                 |
| (4)       | Configuration         |
| (5)       | Directive Extensions  |
| (6)       | Layouts               |
| (7)       | Third-party Libraries |
| (8)       | Internationalization  |
| (9)       | Pages                 |
| (10)      | Routing               |
| (11)      | Services              |
| (12)      | State Management      |
| (13)      | `d.ts` Definitions    |
| (14)      | Utility Classes       |
