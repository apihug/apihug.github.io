## Structure

```
├─assets ①
│  ├─styles
│  │  └─transition
│  └─svg-icon
├─components ②
│  ├─dashboard
│  └─ui
│      ├─accordion
│      ├─alert
├─composables ③
├─config  ④
├─directive ⑤
├─layouts ⑥
│  └─admin
├─lib ⑦
├─locales  ⑧
│  └─langs
│      ├─en-US
│      └─zh-CN
├─pages ⑨
│  ├─dashboard
│  └─example
├─router ⑩
│  └─auto
├─service ⑫
│  ├─api
│  ├─form
│  ├─model
│  └─table
├─stores ⑬
├─types  ⑭
└─utils ⑮
```

| Directory | Content               |
|-----------|-----------------------|
| ①         | CSS Styles            |
| ②         | Component Library     |
| ③         | Hooks                 |
| ④         | Configuration         |
| ⑤         | Directive Extensions  |
| ⑥         | Layouts               |
| ⑦         | Third-party Libraries |
| ⑧         | Internationalization  |
| ⑨         | Pages                 |
| ⑩         | Routing               |
| ⑫         | Services              |
| ⑬         | State Management      |
| ⑭         | `d.ts` Definitions    |
| ⑮         | Utility Classes       |
